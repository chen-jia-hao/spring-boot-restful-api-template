# springboot restful api单表操作模板项目

## 快速开始

1. 安装依赖

   ```shell
   git clone https://github.com/chen-jia-hao/mybatis-generator-lombok-plugin.git
   ```

2. 将依赖安装到本地仓库

    ```shell
    mvn clean install
    ```

3. 克隆本项目

    ```shell
    git clone https://github.com/chen-jia-hao/spring-boot-restful-api-template.git
    ```
